eBPF Talk: 使用 drgn 学习 bpf

2024-06-30
eBPF eBPF Talk
约 1252 字 预计阅读 3 分钟 次阅读

drgn 是 Linux 系统可编程的调试器，基于 Python 脚本简单、便利地读取内核数据结构和变量。

理所当然地，drgn 可以用来学习 bpf，用来 inspect bpf map、bpf program、bpf link 等对象。

P.S. 以下内容基于 drgn PR 提供的功能。
